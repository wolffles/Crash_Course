## PRIMITIVES AND REFERENCES
* In js there are primitive data types and reference data types. 
* PRIMITIVE: referenced by VALUE
  - boolean
  - Number
  - String
  - Null
  - Undefined 
  - Symbol
* REFERENCES: are a Point to memory address
  - Objects {}
  - Arrays
